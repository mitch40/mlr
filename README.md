# MyLittleRouter
Placer le fichier "mlr" dans /usr/local/bin Placer le fichier init.sh où vous le souhaitez (e.g. dans "/tmp", afin qu'il soit effacé automatiquement au redémarrage de la machine.
Exécuter le fichier "init.sh", la machine redémarre automatiquement à l'issue. Utiliser le menu de configuration afin de paramétrer les cartes réseaux.
Bravo vous avez un routeur fonctionnel.
